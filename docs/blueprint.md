# **App Name**: MenuSnap Nutrition

## Core Features:

- Menu Item Input: Allow users to input the name of a menu item.
- Nutrient Input: Allow users to input the AKG (Angka Kecukupan Gizi) values for each menu item.
- Image Capture: Enable users to take a photo of the menu item using their device's camera.
- Mobile-First Design: Optimize the layout and design for mobile devices to ensure a responsive and user-friendly experience.

## Style Guidelines:

- Primary color: A lively teal (#26A69A) to evoke a sense of freshness and health.
- Background color: Very light gray (#F5F5F5) providing a clean and neutral backdrop.
- Accent color: A bright orange (#FF7043) for interactive elements, creating visual interest and guiding user actions.
- Body and headline font: 'Roboto' (sans-serif) for a modern and readable text that suits both headings and body content.
- Employ a material design approach with clear visual hierarchy, using subtle shadows and layering to create depth.
- Focus on a clean, card-based design for displaying menu items and their nutritional information in an organized manner.
- Use simple, flat icons with a touch of vibrant colors to represent different nutrients or categories of menu items.
